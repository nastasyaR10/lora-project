# lora-project

Применение LoRA для адаптации Qwen2.5-0.5B-Instruct к задаче классификации литературных периодов

## О проекте

Проект создан в рамках ВКР. Цель - экспериментально
исследовать эффективность метода LoRA (Low-Rank Adaptation) для дообучения большой
языковой модели Qwen2.5-0.5B-Instruct на задачу классификации фрагментов русской
литературы по пяти историческим периодам («Золотой век», «Реализм», «Серебряный век», «Советский период», «Современная литература») с генерацией
стилистических и содержательных обоснований.

## Структура проекта

•	prepare_data.py – скрипт подготовки набора данных; <br>
•	download_model.py – скрипт загрузки базовой модели из Hugging Face; <br>
•	train.py – скрипт обучения модели методом LoRA; <br>
•	inference.py – скрипт инференса модели; <br>
•	compute_metrics.py – скрипт вычисления метрик на тестовой выборке; <br>
•	server.py – серверная часть приложения на Flask; <br>
•	static/index.html – веб-интерфейс приложения; <br>
•	data/ – папка с файлами train.json и test.json, содержащими обучающую и тестовую выборки; <br>
•	models/lora_adapters – папка с файлами адаптеров, полученных при обучении; <br>
•	output/ - папка с логом и графиками обучения; <br>
•	requirements.txt – список зависимостей.


## Запуск

```cmd
# 1. Клонировать репозиторий
git clone https://github.com/nastasyaR10/lora-project.git
cd lora-project

# 2. Создать и активировать виртуальное окружение
python -m venv venv
venv\Scripts\activate      # Windows

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Запустить веб-сервер
python server.py
```
