# lora-project
Применение LoRA для адаптации Qwen2.5-0.5B-Instruct к задаче классификации литературных периодов

## О проекте

Проект создан в рамках ВКР. Цель - экспериментально
исследовать эффективность метода LoRA (Low-Rank Adaptation) для дообучения большой
языковой модели Qwen2.5-0.5B-Instruct на задачу классификации фрагментов русской
литературы по пяти историческим периодам («Золотой век», «Реализм», «Серебряный век», «Советский период», «Современная литература») с генерацией
стилистических и содержательных обоснований.
